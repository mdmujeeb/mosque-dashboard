Display

Mosque Dashboard uses a Full HD/HD Ready TV as Display. We will provide you with the size of HD/HD Ready TV that you choose while placing order.
Alternately you also have an option to use a Full HD/HD Ready TV that you already own.

Controller

Mosue Dashboard uses a Microcontroller based Controller which has built in intelligence to connect to Cloud through internet, fetch data and display on Screen.
It also plays Bismillah audio when it is time for a Namaz.

Internet Connection

Mosque Dashboard uses a JioFi USB device to connect to internet. This device requires a JIO Telecom SIM card with valid Data balance.
Alternately if your Masjid already has a WiFi network available in range, it can be utilized instead of the JioFi device. In this case, no SIM card is required.

Android App for Update

Mosque Dashboard has an Android app for changing Namaz times and other data on Display.
This App can be installed on any Android Smartphone with Android 4.0 or above.
